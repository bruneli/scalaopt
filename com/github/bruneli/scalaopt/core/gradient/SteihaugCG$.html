    Find pk = zj + tau dj that minimizes mk(pk) = fk - dj pk + 1/2 pkT B pk and satisfies norm(pk) = deltak the trust region size.

    Find pk = zj + tau dj that minimizes mk(pk) = fk - dj pk + 1/2 pkT B pk and satisfies norm(pk) = deltak the trust region size.
