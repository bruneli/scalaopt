package core

Numerical Optimization Algorithms written in Scala.

All optimization algorithms contain a static method named minimize. minimize takes as input a real objective function acting on variables. Variables are represented by any scala collection of Double deriving from Seq like commonly used Array, List, Vector,... The real valued objective function can be any method taking as input a scala collection of Double and returning a Double. The minimize function returns as output a Try[Variables] rather than directly Variables to explicit the fact that all optimization algorithms can fail to find a feasible solution.
