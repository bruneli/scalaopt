object StrongWolfe

Line search algorithm satisfying the strong Wolfe conditions.

The algorithm consists into the search of an acceptable step length alpha along a line. For the strong Wolfe conditions, it requires

  • a sufficient decrease condition: f(alpha) <= f(0) + c1 * alpha * f'(0)
  • a curvature condition: |f'(alpha)| <= -c2 * f'(0) The algorithm is further explained in Chapter 3 of: "J. Nocedal, S.J. Wright, Numerical Optimization, Springer."
